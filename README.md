## Featured Artifacts

### 1.3 — A History of Artificial Intelligence
A visual timeline tracing major AI milestones, breakthroughs, and winters from 1950 through the frontier model era of 2023–2024. Covers the Turing Test, Dartmouth Conference, both AI Winters, Deep Blue, AlexNet, the Transformer architecture, AlphaFold 2, ChatGPT, and the current frontier model race.

---

### 1.4 — ShopBot: Generative AI Customer Service Assistant
A fully functional AI-powered retail chatbot built using the Design Thinking methodology. Includes LLM prompt testing across 3 ChatGPT sessions, STORM AI research, and an end-to-end Botpress build with iterative system prompt engineering.
